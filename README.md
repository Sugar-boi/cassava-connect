# 🌾 Cassava Connect 
**Connecting Farmers and Buyers Directly**

### 🌟 Project Overview
Cassava Connect is a peer-to-peer marketplace designed to eliminate middlemen in the agricultural supply chain. It allows cassava farmers to list their produce and connect directly with bulk buyers, ensuring fair pricing and secure payment tracking.

### 🚀 Key Features
*   **Direct Marketplace:** Farmers can list various cassava types with pricing and location.
*   **Order Tracking:** A status-based tracking system for transparency between parties.
*   **User Authentication:** Secure login for both buyers and sellers.
*   **Mobile-First Design:** Optimized for users in rural areas with high-contrast UI/UX.

### 🛠️ Built With
*   **Frontend:** HTML5, CSS3 (Flexbox & Grid), JavaScript
*   **Backend:** PHP (Server-side logic)
*   **Database:** MySQL (User and product management)
*   **Design Tools:** Figma (UI/UX Prototyping)
